Mediocre chess application written with [Vue](https://vuejs.org/), [TypeScript](https://www.typescriptlang.org/), and [Tauri](https://tauri.app/).

## Usage

```sh
git clone https://github.com/uncenter/rook.git
pnpm install
```

### Web

```
pnpm dev
```

### Desktop (Tauri app)

```
pnpm tauri dev
# or
pnpm tauri build
```
